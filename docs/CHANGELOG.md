# История разработки Survey API

## Чекпоинт 1: Проектирование и старт

### Этап 1: Инициализация проекта
- Создана структура проекта
- Настроен package.json с зависимостями
- Создан .gitignore
- Инициализирован Git репозиторий

### Этап 2: Проектирование базы данных
- Спроектирована ER-диаграмма (7 таблиц)
- Описаны связи между таблицами
- Созданы миграции для PostgreSQL

### Этап 3: Проектирование API
- Составлен список всех эндпоинтов
- Описаны методы, параметры и ответы
- Спроектирована структура запросов/ответов

## Чекпоинт 2: Авторизация и создание опросов

### Этап 4: JWT авторизация
- Реализована регистрация пользователей
- Реализован вход с JWT токенами
- Создан middleware для проверки токенов

### Этап 5: CRUD опросов
- Создание опросов (только для авторов)
- Получение списка опросов с фильтрацией
- Обновление и удаление опросов

### Этап 6: Управление вопросами
- Добавление вопросов к опросам
- Поддержка 3 типов вопросов
- Валидация вопросов и вариантов ответов

## Чекпоинт 3: Жизненный цикл и прохождение

### Этап 7: Жизненный цикл опроса
- Публикация опросов (draft → published)
- Закрытие опросов (published → closed)
- Ограничения на редактирование

### Этап 8: Прохождение опросов
- Получение опроса для прохождения
- Отправка ответов с валидацией
- Защита от повторного прохождения

## Чекпоинт 4: Аналитика и фильтрация

### Этап 9: Аналитика результатов
- Статистика по вопросам с выбором
- Список текстовых ответов
- Экспорт результатов в JSON

### Этап 10: Фильтрация и пагинация
- Фильтрация опросов по статусу
- Пагинация списков
- Сортировка результатов
